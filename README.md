# Алисюнер

Алисюнер — Android-приложение для музыкантов: тюнер, метроном, распознавание аккордов, самоучитель и AI-аранжировщик.

## Что умеет

- Тюнер для разных инструментов и строев.
- Авто-режим с высокой точностью и калибровкой A4.
- Мануальный режим с эталонными звуками струн.
- Метроном с выбором размера такта и регулировкой громкости.
- Распознавание аккордов на 6-струнной гитаре в стандартном строе.
- Самоучитель с библиотекой аккордов и диаграммами грифа.
- Воспроизведение аккордов и струн на реальных акустических WAV-сэмплах.
- AI-аранжировщик с онлайн-режимом и офлайн-фолбэком.

## Интерфейс

- `Тюнер`
- `Метроном`
- `Аккорды`
- `Уроки`
- `AI`

## Тюнер

Поддерживаются:

- Гитара 6 струн, стандартный строй
- Гитара 6 струн, Drop D
- Бас-гитара 4 струны
- Укулеле
- Скрипка
- Альт
- Виолончель

В тюнере можно:

- выбрать инструмент и строй;
- включить авто или ручной режим;
- менять калибровку A4 в диапазоне 430-450 Hz;
- включать режим высокой точности;
- сохранять выбранные настройки между запусками.

## Распознавание аккордов

Детектор аккордов оптимизирован под 6-струнную гитару в классическом строе `E A D G B E`.

Он использует:

- спектральный анализ;
- устойчивое сглаживание результата;
- калибровку A4 из настроек тюнера;
- гитарный частотный диапазон для более точного распознавания.

После записи можно:

- увидеть найденную последовательность;
- транспонировать её вверх или вниз;
- посмотреть аппликатуры на грифе;
- воспроизвести аккорды реальным гитарным сэмплером.

## Самоучитель

В разделе уроков есть:

- библиотека аккордов по тональностям;
- схемы грифа;
- воспроизведение аккордов на реальных сэмплах;
- переход в детальную карточку аккорда.

## AI-аранжировщик

Раздел `AI` умеет:

- генерировать трек через Hugging Face;
- сохранять результат в файл;
- воспроизводить готовую дорожку;
- создавать офлайн-эскиз, если AI-сервис недоступен;
- отменять генерацию.

## Требования

- Android 8.0+ (`minSdk 26`)
- Разрешение на микрофон
- Для AI-режима нужен интернет

## Сборка

### Debug

```bash
./gradlew.bat assembleDebug
```

### Release

```bash
./gradlew.bat assembleRelease
```

### Проверка

```bash
./gradlew.bat lintDebug
```

## APK

Готовые APK обычно сохраняются в папку:

`U:\AI AGENT\ALISUNER CODE\APK`

## Звук

В приложении используются реальные акустические WAV-сэмплы из набора FreePats.

## Лицензии и данные

- Акустические сэмплы: FreePats
- Часть онлайн-функций использует внешние API
